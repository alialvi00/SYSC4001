Instruction:
***How to Compile and Execute****
 1. Download SYSC4001-A3_Submission
 2. Unzip File to an accessible directory
 3. Open two Terminal windows
 4. Use the command: cd <directory where the unzipped folder resides> on both terminal windows
 5. To compile Terminal 1 use the command: gcc RecordKeeper.c -o Record
 6. To compile Terminal 2 use the command: gcc Administrator.c -o Admin
 7. To run Terminal 1 use the command: ./Record 
 8. To run Terminal 1 use the command:./Admin
